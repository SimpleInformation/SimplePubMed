MODULE
------
SI PubMed Fields

Contents of files:
------------------

  * Installation
  * Configuration



REQUIREMENTS
------------
Drupal 7.0


DESCRIPTION/FEATURES
--------------------

SI PubMed Fields Module provides a simple field to import and display scientific publication from the PubMed Database.

How it works
Attach Simple Pumbed Field to any of your node types.
SI PubMed Fields provides a simple text field where you can put in a PubMed ID and SI PubMed Fields provides a content type in installation where you can see the imprted node curresponding to PubMed IDS in the PubMed field.
After saving, PubMed Field will automatically fetch Publication data results from the PubMed database and import it in to a node in .
The formatter will display the result list as links including title, publication date and journal.


Installation:
-------------
1. Copy simplepubmed folder to modules (usually 'sites/all/modules')
   directory.
2. At the 'admin/modules' page, enable the simplepubmed module.


Configuration:
--------------
At the 'admin/config/services/manage-simplepubmed' page can be used for publication from the PubMed Database.
